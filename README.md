# **Teste feito com ReactJS**

## Tecnologias usadas:

- React JS
- Yup
- Formik
- Axios
- Api Rest (feita no mockAPI)

### Como funciona?

- Você deve colocar seu nome, idade, localidade e e-mail nos campos/inputs;
- Os dados serão enviados para a API Rest.
